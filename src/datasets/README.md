# DataSet

1. `train.source` is an example file format for ionize.py if you only have smiles in file. treat as .smi file
2. `CHEMBL_EX_USING_molgpka_atomnum.csv` is an example file format if you have atom number, acid or base, and pKa information. For the 'Atom' column, we used results from MolGpKa. For the 'acd_pka' and 'BasicOrAcid', we got it from ChEMBl database.
